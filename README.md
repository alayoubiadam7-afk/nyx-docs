## 🌐 Common Terms Explained

- **Web3:** The next phase of the web where users control their data and assets directly using blockchain.  
- **Blockchain:** A shared digital ledger that records transactions securely and transparently.  
- **DeFi:** Decentralized finance platforms that let you handle money without banks.  
- **Zero-knowledge proof:** A method to prove something is true without revealing the underlying data.  
- **Cryptography:** Techniques to protect data and communications using codes.  

You do not need to memorize these, but knowing them will help you understand nyx-docs' purpose.
